## Text as confounder

| Title | Description | Code |
|-------|-------------|------|
| [CausalNLP: A Practical Toolkit for Causal Inference with Text](https://arxiv.org/abs/2106.08043) <br> Arun S. Maiya   |   (*also text as treatment*). Describes a toolkit for causal inference with text largely based on meta-learners. Includes support for encoding text as a "controlled-for" variable using traditional BOW features in addition to a PyTorch implementation of Causal Bert (originally from R. Pryzant).
